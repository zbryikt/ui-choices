ui-choices

angularjs binding for bootstrap 3.0 button group
This angular module 'ui-choices' is specific for bootstrap 3.0. Source code can be found at http://github.com/zbryikt/ui-choices/. Here shows some examples about how to use ui-choices in various ways.

Basic Button Group - Single Choice

BananaGraphfruitPitaya
Your Choices:
{{fruit1}}

Source Code

<choices ng-model="fruit1">
    <choice value="banana"> Banana </choice>
    <choice value="graphfruit"> Graphfruit </choice>
    <choice value="pitaya"> Pitaya </choice>
</choices>
